Tutorials

As the main developers of this community project, we understand that sometimes is not easy to achieve exactly what we want. To allow users to get the full benefit of CDF we've prepared a set of documentation - a step by step tutorial from "how to start" to the hidden features of this framework.



1. How to Start

Learn which files to create and what its content.

2. First Dashboard

Select values in different ways and learn how return/forward them.

3. Secure / Prompt

Give a better input to your reports.

4. Charts

Have the freedom to choose how you want to see the data.

5. Templates

Change the look and feel of your dashboards.

6. Timeplots

Interact with timeseries charts.

7. Roles

Create only one dashboard that can be used by colaborators with different roles.

8. Dial & Traffic Charts

Have an immediate idea about the evolution of your business.

9. Geo Dashboard

Enrich how to interact with information through maps.

10. Table & Sparklines

Build tables with visual impact.

11. Drillable Dashboard

Has immediate access and control over your data.

Community Dashboard Framework

Community Dashboard Framework (CDF) is a project that provides an easy to use and powerful dashboard framework on top of the Pentaho BI server. It allows BI developers to quickly and easily create dynamic dashboards that allow users to explore and understand large amounts of data using a variety of charts and tables and then "drill in" to the exact data they want.

CDF brings together the reports, charts, and tables that the Pentaho BI server provides into a cohesive framework that the developer can use to create visually appealing and incredibly useful dashboards using simple HTML and Javascript. New components can be integrated into the framework, and a "MetaLayer" allows developers to save time and prevent copy-and-paste errors by collecting business logic into well organized units that can be shared among dashboards.
